Characters

The characters.json file will describe all the characters that will be participating the Fable Story, the file should contain a list of Characters, each character defined by an Object contained on a global object with a unique key identifying each character.

{  "tonyStark": {    ...  },  "pepperPotts": {    ...  }}

Each character should have the following data,

  • name: The Character Name, will show in each chat bubble. Long names will be truncated with ellipsis...
  • align: left or right
  • avatar: Name of the avatar image on the assets/images folder
  • color: Hex code of the chat bubble color, or a standard css color name (https://www.w3schools.com/cssref/css_colors.asp)
{  "tonyStark": {    "name": "Tony Stark",    "align": "right",    "avatar": "tony.jpeg",    "color": "DarkGoldenRod",  },  "pepperPotts": {    "name": "Pepper Potts",    "align": "left",    "avatar": "pepper.jpeg",    "color": "#DC143C",  }}
